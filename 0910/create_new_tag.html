새로운 태그생성

확인 버튼을 클릭하면 p태그를 생성해서 result1에 추가한다.
ptag = $("<p id='p1' onclick='proc1(this)'>Hello!</p>");
$('#result1').append(ptag);

function proc1(tt){
    $(tt).css('font-size', '+=5px');
}

새로운 태그생성

확인 버튼을 클릭하면 p태그를 생성해서 result2에 추가한다.
ptag = $('<p></p>', {
   'id' : 'p2',
   'text' : 'Hello',
   'click' : function(){
      $(this).css('font-size', '+=5px');
   }
})
